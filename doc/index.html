ServerBuilder

It's a simple NodeJS/Express server builder allowing you to get a working server up in just a few lines.

Install

To install it you need to execute the following:

npm i serverbuilder

Note: Don't forget to use -g, --save, --save-dev if appropriate.

Usage

const app = require('express')(), Server = require('serverbuilder');
let options = {
  name: 'My Server',
  useHttps: false,
  //securityOptions: {} (not needed since HTTPS isn't enabled)
  callback(server) {
    app.set('port', server.port);
  },
  publicIP: true
};

new Server(app, process.env.PORT || 3000, options);
